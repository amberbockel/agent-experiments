# Color Generator App

## When to Activate
1. When the user opens the app and selects the "Generate Colors" option.
2. When the user wants to create a new color palette for their design project.
3. When the user needs inspiration for color combinations.

## Context
The Color Generator App is designed to help users create harmonious color combinations using various color theory models. The app will provide a user-friendly interface for users to select their preferred color theory model and generate random color combinations. The app will also provide examples of design palettes from existing color palette generators as a reference for the user interface and user experience. The app's algorithm will be based on the color theory models and will randomly select colors to apply the chosen model.

## Process
1. The user opens the app and selects the "Generate Colors" option.
2. The user chooses a color theory model (e.g. monochromatic, complementary, analogous, etc.) from a list of available options.
3. The app's algorithm randomly selects a base color and applies the chosen color theory model to generate a set of harmonious color combinations.
4. The app displays the generated color combinations to the user, along with their corresponding hex codes and color theory model used.
5. The user can save, share, or edit the generated color combinations as needed.

## Output Standards
The generated color combinations will be displayed in a format that includes:
* A visual representation of the color palette
* A list of hex codes for each color in the palette
* The name of the color theory model used to generate the palette
* A brief description of the color theory model and its characteristics

The output will be of high quality, with clear and concise language, and will meet the user's expectations for a color generator app.

## Hard Rules
* The app must provide at least 5 different color theory models for the user to choose from.
* The app's algorithm must be able to generate a new set of color combinations each time the user selects the "Generate Colors" option.
* The app must provide a way for the user to save and share their favorite color combinations.
* The app's user interface must be intuitive and easy to use, with clear instructions and minimal clutter.
* The app must be able to generate color combinations that are accessible and usable for users with color vision deficiency.
